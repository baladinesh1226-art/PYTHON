✅ 1. LIST (Most important)
💬 Interview Definition

A list is an ordered, mutable collection that allows duplicate values.

Key Points:
Ordered ✅
Mutable (can change) ✅
Allows duplicates ✅
Indexing possible ✅
Example:
numbers = [10, 20, 30, 20]
numbers[1] = 50
print(numbers)  # [10, 50, 30, 20]
